# CountDawn.py

A simple and lightweight Python command-line application that acts as a countdown timer. You input the total duration in seconds, and the script displays a formatted, ticking countdown ($MM:SS$) directly in your terminal.

---

## Features

* **Dynamic Terminal Update:** The timer updates on a single line without flooding your terminal history.
* **Time Formatting:** Converts raw seconds into a clean `MM:SS` format.
* **Error Handling:** Gracefully handles invalid inputs (like letters or symbols) instead of crashing.

---

## Requirements

* **Python 3.x** installed on your system.
* No external libraries are required (it uses the built-in `time` module).

---

## How to Run

1. **Clone or download** the repository containing `CountDawn.py`.
2. Open your terminal or command prompt and navigate to the folder where the file is located.
3. Run the script using the following command:

```bash
python CountDawn.py
